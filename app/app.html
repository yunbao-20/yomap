APP FEATURES

雲坐標是一款什麼樣的 App？

一個專為雲科大打造的智慧校園應用，融合地圖導航、遊戲化探索與個人化設定
讓「找路」變成一場好玩的校園冒險。

🗺 校園導航系統

內建雲科大 2D 校園地圖，支援即時定位、搜尋建築與設施， 並透過校園路網規劃實際可走的路徑。

🎮 遊戲化探索

將經典吃豆人搬進校園，搭配時間制機制與多款迷你遊戲， 讓逛校園不只是走路，而是一場解任務的冒險。

👤 個人化 & 成就

使用雲科校務信箱登入，支援語言、音效、震動調整， 並透過成就積分解鎖主題與頭像，打造自己的專屬風格。

CAMPUS NAVIGATION

🗺 校園地圖與路網導航

基於自建的校園路網與空間資料庫，雲坐標能顯示雲科專屬的 2D 校園地圖，
不只看得到建築物，還能用關鍵字搜尋教室與設施，並沿著實際道路導航。

  • 搜尋列可輸入「建築物名稱」或「設施關鍵字」，快速找到目的地。
  • 支援即時定位，顯示目前所在位置的藍點，地圖可拖曳與縮放。
  • 透過「地圖資訊」面板，自由開關建築物、ATM、廁所、飲水機、電梯等 POI。
  • 導航路線會跟著校園路網走，不會穿越建築物或水池，路徑更貼近真實動線。

GAME EXPERIENCE

🎮 經典吃豆人 × 迷你遊戲

完成 4 個任務就能通關，過程中還能累積成就與積分。

PERSONALIZATION

👤 個人資料、主題與成就

雲坐標不只是工具，也是一個可以展現自我風格的 App。
從語言、音效到主題與頭像，都能依照自己的喜好調整。

  • 使用雲科校務信箱註冊登入，保障校園社群使用範圍。
  • 支援中文 / 英文切換，一鍵切換介面語言。
  • 音效與震動可獨立開關，符合不同場合需求。
  • 透過完成遊戲與成就，解鎖更多主題顏色與可愛頭像。

HOW IT WORKS

🚀 四步驟開始使用雲坐標

  1. 1. 下載 / 開啟 App

    在擺攤現場掃描 QR Code 或由學校提供下載方式，安裝並開啟「雲坐標」。

  2. 2. 使用校務信箱註冊

    使用 @yuntech.edu.tw 或 @live.yuntech.edu.tw 帳號註冊登入，建立個人資料。

  3. 3. 試用地圖導航

    在地圖頁輸入「系館或教室」，開啟導航並試試看地圖資訊篩選功能。

  4. 4. 進入遊戲解成就

    切換到底部的遊戲分頁，開始吃豆人與迷你遊戲，累積成就、解鎖主題與頭像。
